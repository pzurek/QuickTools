QuickTools for Revit

A set of quick and very simple tools for Autodesk Revit. They are meant to be useful for regular Revit users, but for us are a way of exploring the Revit API and learning how Revit plugins may used to improve users' workflow.

Following tools are included:

Text case tools

Allows to change the text case in all selected text elements simultaneously. If no text elements are selected, the change will be applied to ALL text elements in the project. Available commands are:
  • UPPER CASE
  • lower case
  • Sentence case
  • Title Case

Crop box visibility

Allows to change the visibility of crop boxes on selected views. In case no views were selected, the change will be applied to all views in the project. Available commands are:
  • Hide crop boxes
  • Show crop boxes

Export image

Exports the current view of the graphic area as a PNG file.

Quick wipe

This command deletes all views, scheats and schedules except for the default {3D} view. To use the command, you should first activate the default {3D} view and "Close Hidden" views.

Purge Unnamed RefPlanes

Deletes all unnamed reference planes in the project.

Installation

On Windows 7:
Copy QuickTools.dll and QuickTools.addin into the following folder:
C:\ProgramData\Autodesk\Revit\Addins\2013
Restart Revit. Add-Ins tab should be available in the ribbon. The External Tools dropdown menu will have 8 new commands.
